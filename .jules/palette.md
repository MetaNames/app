## 2024-05-23 - Accessibility Patterns

**Learning:** `CircularProgress` components from `@smui/circular-progress` do not include an `aria-label` by default, requiring manual addition for screen reader context.
**Action:** Always add `aria-label` to `CircularProgress` to explain what is loading.

## 2024-05-23 - Copy-Paste Errors

**Learning:** Copy-pasting code blocks (like list items with images) often leads to incorrect `alt` text being preserved.
**Action:** Double-check `alt` attributes when duplicating image code blocks.
